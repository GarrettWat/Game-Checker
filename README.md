## Run Locally

Clone the project

```bash
  git clone https://github.com/GarrettWat/Game-Checker.git
```

Go to the project directory

```bash
  cd Game-Checker
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
